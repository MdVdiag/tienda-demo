# 🛍️ Tienda Demo

> Catálogo digital con pedidos por WhatsApp, desarrollado para una pyme real.  
> Un proyecto funcional que ayuda a vender más con tecnología sencilla y accesible.

---

## ¿Qué es?

**Tienda Demo** es una aplicación web para la gestión del catálogo de productos y la recepción de pedidos de una pequeña empresa. Ante la decisión de no publicar una tienda online tradicional, se desarrolló esta solución: el cliente navega el catálogo, selecciona sus productos y realiza el pedido directamente por WhatsApp con todos los detalles incluidos automáticamente.

Actualmente en uso por el negocio, aunque no publicada de forma abierta.

---

## Funcionalidades

- 🛒 **Carrito de compra** — añadir, eliminar y gestionar productos
- 📦 **Catálogo por secciones** — organizado por categorías, ampliable
- 🔍 **Filtros de productos** — navegación rápida por el catálogo
- 🖼️ **Galería de productos** — visualización de imágenes
- 📱 **Pedido por WhatsApp** — envío automático del carrito con todos los detalles
- ⚡ **PWA** — instalable en el móvil como si fuera una app nativa

---

## Tecnologías utilizadas

| Tecnología | Uso |
|---|---|
| **HTML** | Estructura de la aplicación |
| **CSS** | Diseño y estilos responsive |
| **JavaScript** | Lógica del carrito y envío por WhatsApp |
| **PWA** | Service Worker para funcionamiento offline e instalación |

---

## Contexto del proyecto

Desarrollado para una pyme del sector de moda infantil (ropa de bebé y complementos). La solución resuelve una necesidad real: disponer de un catálogo digital accesible sin necesidad de gestionar una tienda online completa, utilizando WhatsApp como canal de pedidos, algo ya integrado en el día a día del negocio.

El proyecto está diseñado para ser ampliable a nuevas secciones de producto sin necesidad de grandes cambios.
